# 🇳🇵 Nepal Accounting App

Complete double-entry accounting system for Nepal businesses.

## Features
- ✅ Double-entry accounting (automated)
- ✅ Cash, Bank, eSewa, Khalti support  
- ✅ VAT (13%) compliance
- ✅ Inventory management with COGS
- ✅ Staff advance tracking
- ✅ Financial reports (Trial Balance, P&L, Balance Sheet)
- ✅ Audit-ready with full transaction logs

## Tech Stack
- **Frontend:** Next.js 14 + Tailwind CSS
- **Backend:** NestJS
- **Database:** PostgreSQL + Prisma ORM
- **Auth:** JWT with role-based access

## Quick Start
```bash
cd backend
npm install
npm run dev
